# Authentication Page (Login & Register)

## Project Description

This project is a responsive authentication page that provides **Login** and **Register** functionality using **pure HTML and CSS**.  
The page uses a tab-based interface without JavaScript and is designed with a modern, clean UI suitable for education or web platforms.

---

## Features

- Login and Register forms in a single layout
- Tab switching using **CSS radio buttons**
- Fully responsive design
- Modern UI with gradient background
- Clean and user-friendly form layout
- No JavaScript required

---

## Approach

The page is divided into two main sections:

### Info Panel
- Displays platform branding and description
- Uses a gradient background
- Responsive typography using `clamp()`

### Authentication Panel
- Contains Login and Register forms
- Uses radio buttons to switch tabs
- Only one form is visible at a time using CSS selectors

---

## Technologies Used

- HTML5
- CSS3 (Flexbox)
- Responsive units (`rem`, `vw`, `clamp()`)
